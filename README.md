# MediaContainer
## What's this?
Azure Spatial Analytics の IoT Edge のサンプルなどで利用されている、Virtual RTSP Camera の動作状況を確認するためのツールほか、IoT Edge 上のコンテナを調査するのに便利かもしれないツールをまとめて入れたコンテナです。

## Included tools
- Livemedia Utils
    - openRTSP
    - live555MediaServer
    - playSIP
    - sapWatch
    - vobStreamer
- ffmpeg
- netcat
- iputils-ping
    - ping
- net-tools
    - arp
    - ifconfig
    - netstat
    - rarp
    - route
